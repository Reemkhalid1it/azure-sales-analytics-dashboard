# Azure Sales Analytics Dashboard

## 📌 Project Overview

In this project, I'm building a functional data analytics dashboard using **Power BI**. My main goal is to practice analyzing business datasets and turning raw information into visual insights that actually make sense.

---

## 🎯 What I'm Aiming to Achieve

* Analyze Sales Data: Getting hands-on experience with real-world sales figures.
* Track KPIs: Learning how to monitor **Key Performance Indicators** effectively.
* Spot Trends: Identifying patterns and data movements over time.
* Interactive Design: Presenting everything in a clear, user-friendly, and interactive dashboard.

---

## 🛠 Tools & Tech Stack

* **Power BI**: For building the core visualizations.
* **Microsoft Excel**: For managing and organizing the initial datasets.
* **Data Analysis Basics**: Applying fundamental cleaning and processing techniques.

---

## 📊 Key Features

* **Interactive Dashboards**: Users can filter and explore the data themselves.
* **Visual Storytelling**: Using different charts and graphs to show results.
* **KPI Monitoring**: Visualizing targets and performance metrics.
* **Data Preparation**: Simple data cleaning and transformation (ETL) steps.
